### 产品简介

calculate_JYY 是一款功能强大的图形化科学计算器，对标卡西欧 fx-991 系列。

**主要功能一览**：

| 功能类别 | 支持的功能                        |
| -------- | --------------------------------- |
| 基本计算 | 加减乘除、百分比、括号运算        |
| 科学函数 | 三角函数、对数、指数、阶乘        |
| 方程求解 | 2-4元联立方程、2-4次多项式方程    |
| 矩阵运算 | 4×4矩阵加减乘、转置、求逆、行列式 |
| 向量运算 | 三维向量点积、叉积、模长          |
| 复数运算 | 直角/极坐标、模长、辐角、共轭     |
| 统计分析 | 单变量/双变量统计、6种回归        |
| 进制转换 | 二进制、八进制、十进制、十六进制  |
| 微积分   | 数值导数、定积分                  |
| 科学常数 | 50+ 物理/化学/数学常数            |
| 电子表格 | 公式计算、单元格引用              |
| 数值表格 | 函数值表生成                      |

## 新手快速入门

### 第一步：启动计算器

**Windows 用户**：

1. 找到 `calculate_JYY.exe` 文件
2. 双击该文件
3. 等待计算器界面出现（约2-3秒）

**macOS 用户**：

1. 打开「应用程序」文件夹
2. 找到 `calculate_JYY`
3. 双击启动

**Linux 用户**：

1. 打开终端
2. 进入程序目录
3. 运行 `./calculate_JYY`

### 第三步：做第一个计算

让我们计算 `3 + 4 × 2`：

**操作步骤**：

1. 确认当前在「标准」模式（该标签应高亮显示）
2. 用鼠标点击键盘上的按钮，按顺序输入：

```
点击：【3】→【+】→【4】→【×】→【2】→【=】
```

3. 显示屏将显示：

```
3 + 4 × 2
11
```

💡 **提示**：计算器自动遵循数学运算优先级（先乘除后加减），所以 `3 + 4 × 2 = 11` 而不是 14。

### 第四步：清除并重新开始

- **清除当前输入**：点击【AC】按钮
- **删除最后一个字符**：点击【DEL】按钮

## 基本计算

### 四则运算

#### 加法

**例1**：计算 23 + 45

**操作步骤**：

1. 依次点击：【2】【3】【+】【4】【5】【=】
2. 显示屏显示：
   ```
   23 + 45
   68
   ```

#### 减法

**例2**：计算 100 - 37

**操作步骤**：

1. 依次点击：【1】【0】【0】【-】【3】【7】【=】
2. 显示屏显示：
   ```
   100 - 37
   63
   ```

#### 乘法

**例3**：计算 12 × 15

**操作步骤**：

1. 依次点击：【1】【2】【×】【1】【5】【=】
2. 显示屏显示：
   ```
   12 × 15
   180
   ```

#### 除法

**例4**：计算 144 ÷ 12

**操作步骤**：

1. 依次点击：【1】【4】【4】【÷】【1】【2】【=】
2. 显示屏显示：
   ```
   144 ÷ 12
   12
   ```

### 混合运算

**例5**：计算 2 + 3 × 4

**操作步骤**：

1. 依次点击：【2】【+】【3】【×】【4】【=】
2. 显示屏显示：
   ```
   2 + 3 × 4
   11
   ```

⚠️ **注意**：计算器遵循数学运算优先级，先计算乘法 3×4=12，再计算加法 2+12=14。

**例6**：计算 (2 + 3) × 4

**操作步骤**：

1. 依次点击：【(】【2】【+】【3】【)】【×】【4】【=】
2. 显示屏显示：
   ```
   (2 + 3) × 4
   20
   ```

💡 **提示**：使用括号可以改变运算优先级。括号内的内容会先被计算。

### 小数运算

**例7**：计算 3.14 × 2.5

**操作步骤**：

1. 依次点击：【3】【.】【1】【4】【×】【2】【.】【5】【=】
2. 显示屏显示：
   ```
   3.14 × 2.5
   7.85
   ```

### 负数输入

**例8**：计算 -5 + 3

**操作步骤**：

1. 点击键盘上的【(-)】键（负号键）输入负号
2. 依次点击：【(-)】【5】【+】【3】【=】
3. 显示屏显示：
   ```
   -5 + 3
   -2
   ```

⚠️ **注意**：负号【(-)】和减号【-】是不同的键。负号用于输入负数，减号用于减法运算。

### 百分比计算

**例9**：计算 200 的 15%

**操作步骤**：

1. 依次点击：【2】【0】【0】【×】【1】【5】【%】【=】
2. 显示屏显示：
   ```
   200 × 15%
   30
   ```

**例10**：计算 200 增加 10%

**操作步骤**：

1. 依次点击：【2】【0】【0】【+】【1】【0】【%】【=】
2. 显示屏显示：
   ```
   200 + 10%
   220
   ```

### 幂运算

**例11**：计算 2 的 10 次方

**操作步骤**：

1. 依次点击：【2】【^】【1】【0】【=】
2. 显示屏显示：
   ```
   2^10
   1024
   ```

**例12**：计算 3 的平方根（即 3^0.5）

**操作步骤**：

1. 依次点击：【3】【^】【0】【.】【5】【=】
2. 显示屏显示：
   ```
   3^0.5
   1.732050807568877
   ```

### 使用上次结果 (Ans)

**例13**：先计算 3 + 4，再将结果乘以 2

**操作步骤**：

1. 先计算 3 + 4：
   - 点击：【3】【+】【4】【=】
   - 显示结果：7
2. 再计算结果乘以 2：
   - 点击：【Ans】【×】【2】【=】
   - 显示结果：14

💡 **提示**：Ans 存储了上一次计算的结果，可以直接在新计算中使用。

### 连续计算

**例14**：计算 100 - 50 = 50，然后继续计算 50 + 30 = 80

**操作步骤**：

1. 计算 100 - 50：
   - 点击：【1】【0】【0】【-】【5】【0】【=】
   - 显示结果：50
2. 直接继续输入新的运算（不需要按 AC）：
   - 点击：【+】【3】【0】【=】
   - 显示结果：80

## 函数计算

### 三角函数

计算器支持三种角度单位：

- **DEG**（度）：360度 = 一圈（默认）
- **RAD**（弧度）：2π = 一圈
- **GRAD**（梯度）：400梯度 = 一圈

#### 正弦函数 sin

**例15**：计算 sin(30°)

**操作步骤**：

1. 确认角度单位为 DEG（在设置中查看或修改）
2. 依次点击：【sin】【(】【3】【0】【)】【=】
3. 显示屏显示：
   ```
   sin(30)
   0.5
   ```

**例16**：计算 sin(π/6)（弧度制）

**操作步骤**：

1. 切换角度单位为 RAD（在设置中修改）
2. 依次点击：【sin】【(】【π】【÷】【6】【)】【=】
3. 显示屏显示：
   ```
   sin(π/6)
   0.5
   ```

   - (2, 4)：输入 x=2, y=4
   - (3, 5)：输入 x=3, y=5
   - (4, 4)：输入 x=4, y=4
   - (5, 5)：输入 x=5, y=5
4. 选择回归类型「线性回归」
5. 点击「计算」
6. 显示结果：
   - a = 2.2（截距）
   - b = 0.6（斜率）
   - r = 0.7745966692414834（相关系数）
   - 回归方程：y = 2.2 + 0.6x

### 回归类型说明

| 类型     | 方程             | 适用场景            |
| -------- | ---------------- | ------------------- |
| 线性回归 | y = a + bx       | 数据呈直线趋势      |
| 对数回归 | y = a + b·ln(x)  | 数据增长逐渐放缓    |
| 指数回归 | y = a·e^(bx)     | 数据呈指数增长/衰减 |
| 幂回归   | y = a·x^b        | 数据呈幂函数关系    |
| 逆回归   | y = a + b/x      | 数据呈反比关系      |
| 二次回归 | y = a + bx + cx² | 数据呈抛物线趋势    |

**设置方法**：

   - **DEG**（度）：适用于日常计算、几何问题

   - **GRAD**（梯度）：适用于工程计算

- 180° = π 弧度
- 200 梯度 = 180° = π 弧度

### 显示格式设置

**标准显示 (NORM)**：

- 自动选择最合适的显示方式
- 适合大多数计算

**定点数 (FIX)**：

- 固定显示指定位数的小数
- 适合需要精确小数位数的计算

**科学计数法 (SCI)**：

- 以 a × 10^n 形式显示
- 适合非常大或非常小的数值

**设置小数位数**：

1. 点击 ⚙️ 打开设置
2. 找到「显示格式」选项
3. 选择「FIX」
4. 设置小数位数（0-9）

### 千位分隔符

**开启/关闭方法**：

1. 点击 ⚙️ 打开设置

## 错误信息与解决方法

### 常见错误

| 错误信息        | 中文含义 | 可能原因             | 解决方法                         |
| --------------- | -------- | -------------------- | -------------------------------- |
| Syntax Error    | 语法错误 | 表达式格式不正确     | 检查括号是否配对、运算符是否正确 |
| Math Error      | 数学错误 | 数学运算无意义       | 检查是否除以0、负数开偶次方等    |
| Overflow        | 溢出     | 结果超出计算范围     | 使用更小的数值                   |
| Dimension Error | 维度错误 | 矩阵/向量维度不匹配  | 确保运算对象维度一致             |
| Argument Error  | 参数错误 | 函数参数不在有效范围 | 检查函数输入值                   |
| Undefined       | 未定义   | 变量未赋值就使用     | 先给变量赋值再使用               |

### 详细错误说明与解决

#### Syntax Error（语法错误）

**常见原因**：

1. 括号不配对：`sin(30` 缺少右括号
2. 运算符连续使用：`3 + × 4`
3. 缺少运算数：`3 +`

**解决方法**：

1. 检查括号是否左右配对
2. 检查运算符是否正确连接两个数
3. 确保表达式完整

#### Math Error（数学错误）

**常见原因**：

1. 除以0：`5 ÷ 0`
2. 负数开偶次方：`(-4)^0.5`
3. 对数输入0或负数：`log(0)` 或 `log(-1)`
4. 反三角函数输入超出范围：`sin⁻¹(2)`（应在 -1 到 1 之间）

**解决方法**：

1. 检查除数是否为0
2. 负数开方需要使用复数模式
3. 对数的真数必须为正数
4. 反三角函数的输入必须在有效范围内

#### Overflow（溢出）

**常见原因**：

1. 数值过大：`10^1000`
2. 阶乘过大：`171!`

**解决方法**：

1. 使用科学计数法表示大数
2. 阶乘上限为 170

#### Singular Matrix（奇异矩阵）

**常见原因**：

- 矩阵的行列式为0，矩阵不可逆

**解决方法**：

- 先计算行列式，确认不为0再求逆
- 检查矩阵是否正确输入

## 常见问题

### 基本操作类

**Q1: 如何清除显示屏上的内容？**

A: 点击【AC】按钮清除所有内容，或点击【DEL】删除最后一个字符。

**Q2: 如何输入负数？**

A: 使用键盘上的【(-)】键（负号键），而不是减号【-】。例如输入 -5，点击【(-)】【5】。

**Q3: 如何输入小数？**

A: 点击数字键盘上的【.】键。例如输入 3.14，点击【3】【.】【1】【4】。

**Q4: 计算结果是分数，如何变成小数？**

A: 点击显示屏下方的【S⇔D】按钮，可以在分数和小数之间切换。

**Q5: 如何使用上一次的计算结果？**

A: 点击【Ans】按钮，它会自动调用上一次计算的结果。

### 函数计算类

**Q6: 为什么 sin(30) 不等于 0.5？**

A: 检查角度单位设置。sin(30°) = 0.5，但 sin(30 弧度) ≈ -0.988。确保设置为 DEG（度）模式。

**Q7: 如何计算平方根？**

A: 方法一：输入数值后点击【^】【0】【.】【5】【=】
方法二：使用 sqrt 函数：【sqrt】【(】【数值】【)】【=】

**Q8: 如何计算立方根？**

A: 输入数值后点击【^】【(】【1】【÷】【3】【)】【=】

**Q9: 阶乘最大能算到多少？**

A: 最大支持 170!，171! 及以上会显示 ∞（无穷大）。

### 方程求解类

**Q10: 如何解二元一次方程组？**

A:

1. 点击「方程」标签
2. 选择「联立方程」
3. 选择「2个未知数」
4. 输入系数
5. 点击【=】

**Q11: 方程求解显示"无解"怎么办？**

A: 这表示方程组无解或有无穷多解。检查系数是否正确，或方程组是否矛盾。

### 矩阵向量类

**Q12: 矩阵最大支持多大？**

A: 最大支持 4×4 矩阵。

**Q13: 为什么矩阵求逆出错？**

A: 矩阵必须是方阵且行列式不为0才能求逆。先用 Det() 函数计算行列式。

### 统计计算类

**Q14: 如何删除输入错误的数据？**

A: 在数据编辑器中，选中错误的数据，直接输入新值覆盖，或使用删除功能。

**Q15: 相关系数 r 代表什么？**

A: r 的范围是 -1 到 1：

- r 接近 1：强正相关
- r 接近 -1：强负相关
- r 接近 0：弱相关或无相关

### 设置与界面类

**Q16: 如何切换暗色/亮色主题？**

A: 点击 ⚙️ 设置图标，找到「主题」选项进行切换。

**Q17: 如何切换中英文界面？**

A: 点击 ⚙️ 设置图标，找到「语言」选项进行切换。

**Q18: 计算器反应很慢怎么办？**

A: 尝试关闭其他程序释放内存，或重启计算器。

### 其他问题

**Q19: 如何查看计算历史？**

A: 点击左上角的 ☰ 按钮，打开历史记录面板。

**Q20: 如何报告 Bug 或提建议？**

A: 访问 GitHub 项目页面提交 Issue：https://github.com/JunjieChen0/Calculate/issues
