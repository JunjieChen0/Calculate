# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/), and this project adheres to [Semantic Versioning](https://semver.org/).

## [1.0.0] - 2026-06-22

### Added

- Standard calculator with basic arithmetic operations
- Scientific functions (trig, log, exp, hyperbolic)
- Complex number mode (rectangular and polar)
- Matrix and vector modes
- Equation solver (numerical, quadratic/cubic, linear systems)
- Base-n mode (binary, octal, decimal, hexadecimal)
- Calculus (numerical derivative, integration, summation, product)
- Statistics with 6 regression types and 10+ probability distributions
- Unit conversion, custom functions (f/g/h), variables (A-Z)
- Memory operations (MC/MR/M+/M-/MS)
- History with search and JSON export
- Numerical table and function plotting (Chart.js)
- Spreadsheet (5x20 grid)
- QR code generator and scientific constants library
- Dark/Light theme, Chinese/English i18n
- Keyboard shortcuts, cursor navigation, undo/redo
- KaTeX formula rendering
- Electron desktop app (Windows/macOS/Linux)
- CI/CD pipeline (lint/test/build/release)
- 335 automated tests

### Security

- Electron context isolation and sandbox mode
- Content Security Policy (CSP)
- Store key whitelist with size limits
- Navigation and permission restrictions
- X-Content-Type-Options and X-Frame-Options headers
